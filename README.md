# 💪 POWERFIT GYM

A modern, professional, and fully responsive Gym & Fitness website built using **HTML5**, **CSS3**, and **Vanilla JavaScript**.

Designed with a premium fitness theme featuring modern UI/UX, workout plans, diet plans, trainers, membership pricing, BMI calculator, and contact functionality.

## 🌐 Live Website

https://anilarr.github.io/POWERFIT-GYM/


```bash
Open index.html
```
